Palette: . bg · L body · E eye · e half-eye (blink) · c closed eye

Base idle — ✅ Calm-A (even breathing + decoupled blink)

The always-on idle state. Body cycles rest → up → rest → down at 1200ms per frame. Independent blink channel: random 2-4s intervals, ~20% doubles.

Calm-A · Even breathing ✅

rest → up → rest → down. Even 1200ms each frame. Decoupled blink channel (style A).

Capovolta — ✅ B · Squish + flip

~900 ms total. 6 frames at 150 ms each. Plays once on `octopus finish`, then returns to base.

B · Squish + flip ✅

Anticipation: squashes flat first, then springs up and inverts. Cartoonish bounce.

Moonwalk — ✅ D6 (primary) + E (variant)

D6: 16-col canvas, body ±1 col, legs ratchet ±2 cols, blink at apex. E: body still, leg pairs take turns going up. Both end at the centered base frame for clean handoff.

D6 · Subtle body + ratcheting legs + blink at apex ✅

15 frames. Body shifts ±1 col only. Legs ratchet ±2 (independent of body). At each direction change: blink (drooping → closed → drooping → open with tilt + look).

E · Wave of legs ✅

Body dead still. Each of 4 leg pairs takes a turn going up. Pure leg motion, no body glide. Second moonwalk variant — could trigger on a different verb than D6.

Live sequence — all confirmed animations end-to-end

Cycles through the confirmed animations to test handoffs. Sequence: Calm-A (3.5s, real breathing + blinks) → Capovolta-B → Calm-A → Moonwalk-D6 → Calm-A → Moonwalk-E → Calm-A → loop. Each non-calm animation ends at the centered base frame for clean handoff.

How to read the grid format

Each frame is a 16×14 character grid. . = background, L = body (lavender), E = eye (black), e = half-closed eye (mid-blink), c = closed eye (full blink). The Python TUI reads the same format. Click "Copy grids" on any proposal to grab all its frames as Python-ready string literals.
