Mode switching

H cycles: Full → Compact → Slim → Full. Override is per-session (not persisted). Auto-default by terminal width: ≥140 cols → Full, 80–139 → Compact, <80 → Slim. Resize re-evaluates only when the user hasn't manually overridden — explicit choice sticks until quit.
